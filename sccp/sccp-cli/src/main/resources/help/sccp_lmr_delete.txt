Name
	sccp lmr delete

SYNOPSIS
	sccp lmr delete <id>

DESCRIPTION
	This command is used to delete a Long Message Rule.

PARAMETERS

	Standard Parameters
                                             
	<id>		-	The identifier of the Long Message Rule that is being
				deleted.

EXAMPLES
	sccp lmr delete 1 

	The above command will delete the Long Message Rule identified by the number '1'.
