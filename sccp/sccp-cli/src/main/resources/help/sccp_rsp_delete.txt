Name
	sccp rsp delete

SYNOPSIS
	sccp rsp delete <id>

DESCRIPTION
	This command is used to delete a Remote Signaling Point.

PARAMETERS

	Standard Parameters
                                             
	<id>		-	The identifier of the Remote Signaling Point that is being
				deleted.

EXAMPLES
	sccp rsp delete 1 

	The above command will delete the Remote Signaling Point identified by the 
	number '1'.
