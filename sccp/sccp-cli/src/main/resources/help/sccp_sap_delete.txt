Name
	sccp sap delete

SYNOPSIS
	sccp sap delete <id>

DESCRIPTION
	This command is used to delete a previously defined Service Access Point.

PARAMETERS

	Standard Parameters

	<id>		-	The id of the SAP that is being deleted.  

EXAMPLES
	sccp sap delete 1
	
	The above command will delete the SAP identified by the number '1'.
