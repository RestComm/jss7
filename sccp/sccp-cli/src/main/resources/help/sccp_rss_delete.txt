Name
	sccp rss delete

SYNOPSIS
	sccp rss delete <id>

DESCRIPTION
	This command is used to delete a Remote Sub-System.

PARAMETERS

	Standard Parameters
                                             
	<id>		-	The identifier of the Remote Sub-System that is being 
				deleted.

EXAMPLES
	sccp rss delete 1 

	The above command will delete the Remote Sub-System identified by the number '1'.
