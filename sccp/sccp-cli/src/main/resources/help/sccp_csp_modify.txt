Name
	sccp csp modify

SYNOPSIS
	sccp csp modify <id> <spc>

DESCRIPTION
	This command is used to modify the values of a Concerned Signaling Point Code 
	previously defined.

PARAMETERS

	Standard Parameters
                                              
	<id>			-	Identifier of the Concerned Signaling Point Code
					to be modified.

	<spc>			-	The Remote Signaling Point Code, which will be 
					notified.
